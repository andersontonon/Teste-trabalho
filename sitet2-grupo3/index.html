ProBuilder - Process Builder V1.0
Este documento faz a especificação de um processo de software no qual resulta em um produto de software desenvolvido. Este produto de software será um programa de gerenciamento de estoque para itens de mercado. Inclui as atividades, artefatos e papéis bem como a interligação entre esses elementos, além dos templates e guidances que auxiliam na realização do mesmo.
